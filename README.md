# NomadInjector ‚ú®

C# WPF LoadLibraryA Injector and Process Diagnostic Tool.

## Key Features & Benefits üîë

* **DLL Injection:** Inject DLLs into running processes using the **LoadLibraryA** technique.
* **Process Diagnostic:** Provides detailed information about running processes and their loaded modules.
* **User-Friendly Interface:** Intuitive WPF interface for easy navigation and operation (Light Theme, Emerald Green Accent).
* **Module Viewer:** Inspect loaded modules within a process.
* **Settings Persistence:** Saves injection settings for future use.
* **Credits:** Includes a credits window acknowledging contributors and resources.

## Installation & Setup Instructions ‚öôÔ∏è

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/Deportando/NomadInjector.git](https://github.com/Deportando/NomadInjector.git)
    ```

2.  **Open the solution:**
    Open `NomadInjector.sln` using Visual Studio.

3.  **Build the project:**
    Build the `NomadInjector` project within Visual Studio.

4.  **Locate the executable:**
    The compiled executable (`NomadInjector.exe`) will be located in the `NomadInjector/bin/Release` directory, depending on your build configuration.
    
5.  **Run as Administrator:** The application requires administrator privileges for injection.
